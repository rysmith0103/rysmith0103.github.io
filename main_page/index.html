Assignment 1: Basic HTML

This assignment covers the basics of HTML structure, including elements such as headers, paragraphs, lists, and links.

Assignment 2: Basic CSS

This assignment focuses on styling HTML elements using CSS, including properties for color, padding, margins, and borders.

Assignment 3: Page Layout

This assignment introduces page layout techniques, including the use of Flexbox to organize content effectively.

Assignment 4: Recreate CSS Page

This assignment is meant to recreate the "Graduate School Admissions" page using HTML/CSS, placeholder images, and Lorem Ipsum text. The focus is on accurate layout replication using Flexbox and submitting the project link on time

Assignment 5: JavaScript Buttons, Functions, and More

The project aims to create a responsive web page with a 3-column layout, interactive features like click counting, image refreshing, and slider-based movement, using HTML, CSS, and JavaScript.

Assignment 6: If Statements

The goal of this project is to create a responsive menu bar with toggle functionality and stylish design. Also a goal is to learn how if statements and other JavaScript properties work.

Assignment 7: JavaScript: Loops

The goal of this project is to implement dynamic content generation and event handling through user interactions, while applying modern JavaScript standards, error handling, and responsive design principles.

Assignment 8: JavaScript: Arrays

The goal of the project is to create a web page that dynamically displays a set of images from an associative array, with custom titles and descriptions, and implements interactivity by displaying image details when clicked, using modern JavaScript practices and proper code formatting.

Assignment 9: JavaScript: Classes

The primary learning goal of this project is to create a dynamic list of items using JavaScript classes and arrays, display the content on a webpage using formatted HTML sections, and implement interactive features such as modal dialogs for detailed item views, emphasizing the use of JavaScript classes and DOM manipulation.

Assignment 10: Parsing JSON

The primary goal of this project is to learn how to fetch and parse JSON data asynchronously, display it dynamically on a webpage, and implement CSS overlays for enhanced user interaction.
